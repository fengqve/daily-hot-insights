🐙 代码世界（GitHub + 技术）

  • openclaw/openclaw | AI/本地化助手框架 | 2026年黑马项目，被称为"个人 AI 助手框架的 break-out star"。完全在用户本地运行，连接 AI 模型与 50+ 集成服务（WhatsApp、Telegram 等），支持浏览器控制、shell 命令执行、代码编写和智能家居控制。其核心亮点是能自行编写新技能扩展自身功能，1 月底病毒式传播后已斩获 21 万+ star。**洞察**：OpenClaw 的自举能力（自己写技能）是今年 Agent 框架最值得关注的技术边界探索——如果一个框架能扩展自己，整个生态的进化速度将发生质变。
  • ollama/ollama | Go/本地 LLM 运行时 | 简化本地大模型运行，用一条命令即可拉起 Llama、DeepSeek、Mistral、Gemma 等多种模型，抹平 Python 环境复杂度。提供 macOS 和 Windows 桌面应用。**洞察**：Ollama 已从极客玩具变成 AI 应用基础设施，其"一条命令运行模型"的理念正在重塑本地 AI 部署的门槛，预计今年渗透率将持续攀升。
  • langchain-ai/langchain | Python/AI Agent 框架 | AI Agent 构建的基础框架，提供链、Agent、记忆、检索、工具调用和多 Agent 编排的模块化组件。LangGraph 进一步扩展了对复杂有状态 Agent 工作流的支持。**洞察**：LangChain 已成 AI Agent 领域的"操作系统"级基础设施，大多数新框架都在集成或借鉴它，但复杂度管理仍是最大槽点。
  • google-gemini/gemini-cli | TypeScript/Google AI CLI | Google 开源的命令行 AI 代理工具，将 Gemini 直接集成到终端，支持代码执行、文件操作和多步骤任务。**洞察**：Gemini CLI 的出现意味着 Google 正式加入 AI 编码 Agent 的终端争夺战，与 Claude Code、Copilot Workspace 形成三足鼎立格局。
  • ruc-datalab/DeepAnalyze | Python/数据科学 Agent | 首个代理级 LLM，可自动分析大型数据集并生成专业分析报告，实现从数据到洞察的端到端自动化。**洞察**：数据科学是 AI 替代"白领工作"最直接的场景之一，DeepAnalyze 这类工具让"一个人就是一个数据分析团队"成为可能。
  • BerriAI/litellm | Python/多模型网关 | 提供 OpenAI 兼容 API 格式，调用 100+ LLM（包括 Bedrock、Azure、VertexAI、Anthropic），并具备成本跟踪、限流、日志和负载均衡功能。**洞察**：多模型共存时代，litellm 这类"统一网关"解决了企业同时使用多个供应商模型的工程难题，是今年企业 AI 架构的标配中间件。
  • infiniflow/ragflow | Python/RAG 引擎 | 企业级 RAG（检索增强生成）引擎，专注于让 AI 访问特定领域数据，支持知识库构建和精排检索。**洞察**：RAG 仍是企业私有知识落地的最优解，RAGFlow 以"开箱即用"降低企业部署门槛，在金融、医疗、法律领域有强需求。
  • **洞察**：本期 GitHub 主题高度集中于** Agent 基础设施**（LangChain、OpenClaw）和**本地 AI 运行时**（Ollama、litellm）两条主线。延续昨天 deer-flow 等多 Agent 框架的热度，但今天出现了一个新变量：**OpenClaw 的自举能力**和 **Gemini CLI 的终端争夺**，预示 Agent 框架正从"构建工具"进化到"接管工作流"。
